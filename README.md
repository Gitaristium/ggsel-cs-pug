## Доступные команды

### Установка

Обязательно установите зависимости:

```
npm install
```

## Сервер разработки

Запустите сервер разработки на `http://localhost:3000`:

```
gulp
```

### Продакшен

Соберите приложение для продакшена:

```
gulp build
```
